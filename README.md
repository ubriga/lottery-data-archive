# 🎲 Israeli Lottery Data Archive

**ארכיון תוצאות הגרלות ישראליות ממקורות רשמיים בלבד**

## 📋 סקירה

מאגר נתונים היסטורי של הגרלות בישראל, עם דגש על:
- שקיפות מקור לכל רשומה
- אימות דומיין/מקור
- עדכונים אוטומטיים באמצעות GitHub Actions

> נכון לעכשיו, תהליך ה־ETL מופעל רק עבור משחקי מפעל הפיס שבהם יש ארכיון רשמי נגיש לציבור.

## 📂 מבנה הנתונים

```
datasets/
├── _meta/
│   ├── coverage.json      # כיסוי נתונים - תאריכים, חוסרים
│   └── sources.json       # מקורות רשמיים לכל משחק
├── pais/
│   ├── lotto/
│   └── chance/
└── sportoto/
    ├── winner/
    └── toto/
```

## 🔄 עדכניות

העדכון נרשם ב־[`datasets/_meta/coverage.json`](datasets/_meta/coverage.json).

## ⚠️ הערה חשובה

הגרלות הן אקראיות; ניתוח היסטורי אינו מהווה הבטחה לתוצאות עתידיות.
